Personality Database

High-Fidelity Daily Activity Visualization & Metrics Compilation.

System Architecture

A comprehensive, developer-centric dashboard designed to centralize tasks, scheduling, coding resources, and analytics into a single local environment.

Portability Layer

While initially built with Node.js, Express, and SQLite, the current codebase implements a clever portability layer (via the DB object) that mocks database requests using the browser's localStorage. This allows the entire application to run seamlessly as a static site (e.g., on GitHub Pages) while remaining structured like a RESTful API application.

Tech Stack & Libraries

Frontend
HTML5, Vanilla JavaScript, and Tailwind CSS (via CDN) for styling.
Typography
Lucide Icons, 'Inter' for UI, and 'Fira Code' for code sections.
Libraries
marked.js for Markdown parsing, Chart.js for rendering productivity analytics.
External APIs
Groq API: Powers "Zen Assistant" AI (llama-3.1-8b).
Tavily API: Live internet search capabilities for the AI.
GitHub API: Fetches profiles, repos, and commit events.
wttr.in: Real-time weather tracking.

Core Modules

01

Security & Auth

Guarded by an access protocol interface requiring a hardcoded "Access Protocol Key". Users can provide a Groq API Key at login to initialize AI features securely via localStorage.

02

Zen Intelligence Assistant

A floating, LLaMA-powered AI that acts as the system brain. It provides proactive briefings, understands context via system prompts, uses Tavily for web searches, and autonomously executes command protocols (e.g., [ADD_TASK]).

03

Task Management

A dual-system featuring a Priority Tasks to-do list and a Core Scheduler—a custom-built, interactive monthly calendar grid for tracking "Objectives".

04

Dashboard & Focus Engine

The main landing screen features a Real-time HUD, a visual Pomodoro focus timer with an SVG circular progress bar, and a live Activity Stream logging tasks and code pushes.

05

GitHub Intelligence

Syncs with GitHub using a Personal Access Token. It automatically pulls repository data, filters PushEvents into the dashboard's activity feed, and renders a dynamic contribution graph.

06

Code & Data Vaults

Code Vault: A snippet manager with language-specific tagging (JS, Python, Rust, SQL).
Data Vault: A repository for external resources like videos and markdown notes.

07

Zen Writer

A distraction-free, highly customizable Markdown editor. Features a dual-pane setup with live formatting preview, directly saving documents into the Data Vault.

08

Analytics & Health

Renders a "Weekly Completion Velocity" chart and actively pings the Groq and GitHub APIs every 5 seconds to ensure credential validity and network stability.

Design System

The UI follows a strict Dark Mode / Cyberpunk / Glassmorphism aesthetic. It utilizes deep black backgrounds with ambient blurred video loops, leveraging translucent backgrounds, backdrop-filter: blur(), and subtle bright borders. Neon accents are driven by a fruit gradient (#ff0080 to #7928ca) for key AI triggers, complemented by physics-based CSS transitions.
